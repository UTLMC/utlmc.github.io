    Home

    This website is an editor for uploading LMC data, viewing it, making changes, and exporting it for use in the LMC website. Data files must be put in the same folder as index.html and be named data.js to be recognized. Please remember to export your changes before exiting or refreshing this page, as they are NOT stored.

    The editor was created to shield you from interacting with the inner HTML/CSS/Javascript code in the website itself.
