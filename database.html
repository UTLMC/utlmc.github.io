    Instructions

    This website is an editor for handling LMC data for use in the LMC website. If what you need to change is not documented here, contact one of the developers to make the edit.

    Changes made here are not automatically reflected in the website. You must export the data as a file and overwrite the existing data file manually on Github. The data file must be called data.js and placed in the same folder as this page (database.html). Make sure to keep backups when making large changes, as undo/auto-saving is not implemented.

    CSV to JS

    The Google Sheets database is convenient but doesn't enforce consistent naming conventions, so try to avoid editing data there. It does however contain links to member Discords. This website deliberately avoids extracting Discords of non-execs for the current year to avoid revealing more personal information than strictly necessary (current year = the latest year of any tag that has a year in it).

    To export data from Google Sheets for the website, import each page separately as a CSV and upload them here. If you want to include a person's Discord links, they must be manually added to the "Personal Links" column.
